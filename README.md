❤️ Heart Disease Prediction by Machine Learning

A professional Machine Learning API that predicts the likelihood of heart disease using patient health information. This project was built with a focus on performance, scalability, and deployment, making it suitable for educational portfolios and real-world API demonstrations.

---

🚀 Features

- Machine Learning–based heart disease prediction
- FastAPI REST API
- Interactive Swagger UI documentation
- Probability score for predictions
- Health check endpoint
- Production-ready deployment
- JSON request and response
- Easy integration with web and mobile applications

---

🛠️ Technologies Used

- Python
- FastAPI
- Scikit-learn
- CatBoost
- NumPy
- Joblib
- Uvicorn

📥 Installation

Clone the repository:

git clone https://github.com/your-username/heart-disease-prediction.git

Move into the project directory:

cd heart-disease-prediction

Install dependencies:

pip install -r requirements.txt

Run the application:

uvicorn app:app --reload

🔍 API Endpoints

GET /

Returns basic information about the API.

GET /health

Checks whether the API and model are running successfully.

POST /predict

Predicts the likelihood of heart disease from patient health data.
